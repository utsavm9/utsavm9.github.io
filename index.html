Mathematical Expression Simplifying Program

This program is a term-rewriting program which repeatedly applies all the available rules to the expression being simplified. The rules are maintained in a rule book, which can be edited by the user. The program also comes with the capabilities of allowing the user to add and delete rules.

Cellular Automata Simulating Program

I developed this program to understand how animation and graphic-like abilities were achieved in the era of command-line interfaces. The program simulates the two simplest cellular automata: Conway's Game of Life and Wolfram's Elementary Cellular Automata.

The algorithm implemented to achieve animation re-prints the entire screen for each generation. Some of the challenges we overcome in this project are:

  • Turning rectangle MS-DOX "grids" into a sqaure grid.
  • Matrix operations
  • Window-like user interface in commnad-line environment

Coordinate Geometry: Graph Plotter

Line Plotter is a math educational app that plots a quadratic function on the screen. It is a useful app for students learning Coordinate geometry as it helps them to visualize the graph and better understand the concepts.

The app has the following features:

  • Plot quadratic and linear equations around the origin
  • The facility to save the graphs for future reference
  • Change the scale and the color of the graph
  • The axes show numerical markings which adapt according to the scale
